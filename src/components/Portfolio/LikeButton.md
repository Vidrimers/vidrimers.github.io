# LikeButton Component

Компонент кнопки лайка для проектов портфолио с интеграцией в систему лайков.

## Описание

LikeButton - это интерактивный компонент, который позволяет пользователям ставить лайки проектам в портфолио. Компонент автоматически синхронизируется с backend API и сохраняет состояние в localStorage для улучшения пользовательского опыта.

## Использование

```jsx
import LikeButton from './components/Portfolio/LikeButton';

// Базовое использование
<LikeButton projectId="pet-1" />

// С дополнительными CSS классами
<LikeButton 
  projectId="layout-2" 
  className="custom-like-button" 
/>
```

## Props

| Prop | Тип | Обязательный | По умолчанию | Описание |
|------|-----|--------------|--------------|----------|
| `projectId` | `string` | ✅ | - | Уникальный ID проекта (например, 'pet-1', 'layout-2') |
| `className` | `string` | ❌ | `''` | Дополнительные CSS классы |

## Состояния компонента

### 1. Обычное состояние
- Пустое сердечко (♡)
- Отображается количество лайков
- При наведении появляется hover эффект

### 2. Лайкнутое состояние
- Заполненное сердечко (♥)
- Розовый градиентный фон
- Белый текст

### 3. Состояние загрузки
- Показывается "..." вместо количества лайков
- Кнопка заблокирована
- Анимированные точки загрузки

### 4. Состояние ошибки
- Красноватый фон
- Показывается "—" вместо количества
- Кнопка заблокирована

## Анимации

- **Hover эффект**: Подъем кнопки и изменение цвета
- **Клик анимация**: Пульсация кнопки при клике
- **Анимация сердечка**: Увеличение сердечка при переключении лайка
- **Появление**: Плавное появление при первой загрузке

## Доступность (Accessibility)

- Полная поддержка клавиатурной навигации
- ARIA-атрибуты для скринридеров
- Высокий контраст для пользователей с ограниченными возможностями
- Уважение к настройкам `prefers-reduced-motion`

## Адаптивность

- Автоматическая адаптация под мобильные устройства
- Увеличенная область касания на touch-устройствах
- Поддержка темной темы через `prefers-color-scheme`

## Интеграция с API

Компонент использует хук `useLikes` для взаимодействия с backend:

```javascript
// Автоматические API вызовы:
GET /api/likes/{projectId}     // Получение количества лайков
POST /api/likes/{projectId}    // Добавление/удаление лайка
```

## Кэширование

- Лайки кэшируются в localStorage на 5 минут
- Состояние лайка (лайкнуто/не лайкнуто) сохраняется постоянно
- При ошибках сети используются кэшированные данные

## Примеры использования

### В карточке проекта

```jsx
<div className="project-card">
  <img src={project.image} alt={project.title} />
  <h3>{project.title}</h3>
  <p>{project.description}</p>
  
  {/* Кнопка лайка в правом верхнем углу */}
  <div className="like-button-container">
    <LikeButton projectId={project.id} />
  </div>
</div>
```

### Стилизация контейнера

```css
.like-button-container {
  position: absolute;
  top: 10px;
  right: 10px;
  z-index: 10;
}

/* Показывать только при наведении на карточку */
.project-card:hover .like-button-container {
  opacity: 1;
}

.project-card .like-button-container {
  opacity: 0;
  transition: opacity 0.2s ease;
}
```
