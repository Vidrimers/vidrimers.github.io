# Footer Component

Компонент подвала сайта с контактной информацией и ссылками на социальные сети.

## Использование

```jsx
import Footer from './components/Layout/Footer/Footer';

function App() {
  return (
    <div>
      {/* Другие компоненты */}
      <Footer />
    </div>
  );
}
```

## Особенности

- Использует LanguageContext для мультиязычности
- Отображает контактную информацию (email)
- Ссылки на социальные сети (LinkedIn, Telegram)
- Адаптивный дизайн для всех устройств
- CSS Modules для изоляции стилей

## Структура

- `Footer.jsx` - React компонент
- `Footer.module.css` - стили компонента
- Использует переводы из `src/data/translations/`

## Переводы

Компонент использует следующие ключи переводов:

```json
{
  "footer": {
    "title": "Контакты",
    "text": "Связаться со мной можно по данным ссылочкам",
    "sendMessage": "Отправить сообщение",
    "findMe": "Меня можно найти и добавить",
    "onSocial": "На линкедине и в телеграме",
    "thanks": "СПАСИБО :-)"
  }
}
```

## Стили

Компонент использует CSS Modules с адаптивным дизайном:
- Desktop: полный размер
- Tablet (≤768px): уменьшенные размеры
- Mobile (≤480px): компактный вид

## Зависимости

- React
- LanguageContext
- CSS Modules
