# ScrollToTop Component

## Описание

Компонент кнопки "Вверх" для быстрой прокрутки страницы к началу.

## Функциональность

- **Автоматическое показ/скрытие**: Кнопка появляется при прокрутке вниз более чем на 300px
- **Плавная прокрутка**: Использует `smooth` behavior для плавной анимации
- **Мультиязычность**: Текст кнопки меняется в зависимости от выбранного языка (ru/en)
- **Адаптивность**: Корректно отображается на всех устройствах

## Использование

```jsx
import ScrollToTop from './components/Layout/ScrollToTop/ScrollToTop';

function App() {
  return (
    <div>
      {/* Ваш контент */}
      <ScrollToTop />
    </div>
  );
}
```

## Стили

Компонент использует CSS Modules для изоляции стилей:
- `ScrollToTop.module.css` - основные стили
- Адаптивные брейкпоинты: 768px (tablet), 480px (mobile)

## Зависимости

- React (useState, useEffect, useContext)
- LanguageContext для мультиязычности
- Изображение: `/assets/img/ico/arrow-up.png`

## Критерии приемки

- ✅ ScrollToTop компонент создан
- ✅ Кнопка показывается при скролле вниз (>300px)
- ✅ Кнопка скрывается вверху страницы (<300px)
- ✅ Прокрутка вверх работает (smooth scroll)
- ✅ Стили применены и адаптивны
